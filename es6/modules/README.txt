To run this demo from a command-line
  traceur main6

The web version of this demo requires traceur-runtime.js.
If traceur was installed with "npm install -g traceur"
then this file can be copied from `npm -g root`/traceur/bin.

To build this demo
  traceur --script main6.js --out gen/main.js --sourcemap

To run this demo in a modern browser
  open index.html
